## 变更

- 新增 `ops/scripts/local/import-github-project-cards.ps1`。
- 支持输入一个或多个 GitHub 仓库 / 子目录 URL，生成 FastGPT 项目推荐卡片。
- 支持 `-UrlsFile` 批量 URL、dry-run 预览、`-Import` 导入、`-SkipExisting` 按来源去重。
- 对 `awesome-llm-apps` 已有项目优先复用本地 v2 高质量中文卡片。
- 对其它 GitHub 项目基于 metadata、README、文件树生成通用卡片；GitHub API 限流时降级读取 README。
- NextChat / ChatGPT Web UI 类项目优先输出中文卡片，并归类为 `LLM 聊天前端`。
- 更新项目文档中的脚本操作入口。
- 更新服务器 SSH 别名说明为 `ssh sub2api-cf`。
