## 变更

- 将项目整合为单仓库结构：`ops/` 保存运维文档脚本，`fastgpt/` 保存 FastGPT 私有二开源码，`artifacts/` 保存本地临时产物。
- 修复本地启动脚本迁移后的路径问题，支持从仓库根目录调用 `ops/scripts/local/*.ps1`。
- 修复本地 Docker override 的 FastGPT、MinIO 端口覆盖，避免 `STORAGE_EXTERNAL_ENDPOINT` 指向未映射端口导致初始化失败。
- 本地轻量启动移除对 sandbox、MCP、OpenSandbox、volume-manager 的依赖，并在启动时清理孤儿容器。
- 补充 `.gitignore`，忽略 `.tgz`、`.vercel/`、本地运行态、依赖、构建缓存和临时产物。
- 继续保持模型提供商、模型名称、Base URL、API Key 和 Embedding 配置只通过 FastGPT UI 维护，不进入代码和文档配置。
