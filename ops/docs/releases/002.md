## 变更

- 在 FastGPT 默认引用模板中增加 `sourceUrl` 字段，值来自 `sourceId`，用于让回答正文生成 Markdown 可点击来源链接。
- URL 目录导入默认使用正文选择器，优先提取 `article`、正文容器和 `main`，减少页面导航、侧边栏等噪音。
- 选择器支持逗号分隔后按首个命中的选择器提取，避免父子选择器同时命中造成正文重复。
- URL 目录预览过滤首页、分类、标签、归档、搜索、feed、rss、静态资源等非正文页面。
